🍏 Guide Apple Silicon

1

Télécharger l'image ARM64

Cliquez sur le bouton ci-dessus ou utilisez:

curl -O https://dl.pixelos.org/iso/pixelos-latest-arm64.img
2

Créer la clé USB bootable

# Identifier le disque
diskutil list
# Écrire l'image (attention: efface tout le disque!)
sudo dd if=pixelos-latest-arm64.img of=/dev/rdisk2 bs=1m status=progress
3

Démarrer sur Apple Silicon

Méthode 1 — Boot natif:

1. Éteindre le Mac
2. Maintenir POWER enfoncé
3. Choisir "EFI Boot"

Méthode 2 — Asahi Linux (dual-boot):

curl -sL https://alx.sh | sh
# Puis installer PixelOS dans la partition

Méthode 3 — UTM (VM):

1. Installer UTM depuis https://mac.getutm.app/
2. Nouvelle VM → OpenBSD → Choisir l'image
4

Première connexion

Connectez-vous en SSH ou via le navigateur:

ssh root@ (mot de passe: pixelos2024)
http://:9999 (interface web)
pixelos federation discover
pixelos portal join --nickname "Mon Mac" --country "FR"

🖥️ Guide Intel Mac / PC

Procédure identique mais avec l'ISO AMD64. Le démarrage se fait avec la touche Option (⌥) sur Mac, ou F12/DEL sur PC.

📜 Vérification des signatures

# Vérifier l'intégrité du téléchargement
shasum -a 256 pixelos-latest-*.img
# Comparer avec le fichier .sha256 fourni

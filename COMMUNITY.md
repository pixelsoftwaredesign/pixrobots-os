# 🤝 Rejoignez l'écosystème Pixel OS

Pixel OS construit un Internet agricole décentralisé — des fermes connectées, des robots autonomes et des communautés souveraines, sans dépendre d'aucun cloud.

Nous venons de publier le **SDK pixcore-android** et 4 applications Android. Il manque **vos idées, votre code, votre temps**.

---

## 📱 Apps à construire

| App | Description | Compétences clés | Statut |
|-----|-------------|------------------|--------|
| **PixOS Livestream** | Streaming live P2P (WebRTC) — caméras ferme, robots, événements communautaires | Kotlin, WebRTC, Matrix | 🔴 À faire |
| **PixOS Office** | Suite bureautique collaborative (Docs, Sheets, Slides, Base) — CRDT, stockage IPFS | Kotlin, CRDT, Compose Canvas | 🔴 À faire |
| **PixOS Phone** | Dialer GSM + VoIP Matrix — contacts, historique, chiffrement | Kotlin, TelecomManager, Matrix VoIP | 🔴 À faire |
| **PixOS Mail** | Messagerie décentralisée (Matrix) — boîte mail, pièces jointes IPFS | Kotlin, Matrix SDK | 🔴 À faire |
| **PixDomotics** | Domotique IoT — capteurs, actionneurs, scénarios | Kotlin, MQTT, BLE | 🔴 À faire |
| **PixEmbedded** | Flashage et configuration de microcontrôleurs | Kotlin, USB Serial | 🔴 À faire |

## 🔧 Compétences recherchées

- **Kotlin / Jetpack Compose** — développement Android
- **WebRTC** — streaming pair-à-pair
- **Matrix** — messagerie décentralisée
- **Web3 / Ethereum** — wallet, smart contracts, ENS
- **IPFS / Filecoin** — stockage décentralisé
- **BLE / IoT** — capteurs agricoles
- **UI/UX Design** — interfaces rurales, accessibles
- **Traduction** — FR, EN, ES, PT, AR

## 🏆 Récompenses

Les contributeurs reçoivent des **BITROOT** (BRT) en fonction de la qualité et de l'adoption de leur travail.

## 🚀 Comment contribuer

```bash
# 1. Forker le dépôt principal
git clone https://github.com/pixelsoftwaredesign/pixelos-agricol.git

# 2. Créer votre app en important le SDK pixcore-android
cd pixelos-agricol
# Voir pixos/ ou pixos-messenger/ pour exemple

# 3. Builder
./gradlew assembleDebug

# 4. Partager l'APK ou soumettre une PR
```

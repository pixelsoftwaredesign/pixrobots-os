# AgriculApp (mobile)

Application mobile Flutter alternative pour Pixel OS. Version cross-platform du client Pixel OS.

## Fonctionnalités

- Dashboard temps réel
- Capteurs (MQTT)
- Robots et missions
- Zones et alertes
- Authentification PixKey (token + QR code)
- API REST

## Stack

- Flutter / Dart
- MQTT (capteurs temps réel)
- Provider (état)

## Build

```bash
flutter pub get
flutter build apk
```
